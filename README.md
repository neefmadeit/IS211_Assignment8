How to Run /n
Normal game (human vs human)
python3 pig.py

Human vs computer
python3 pig.py --player1 human --player2 computer

Computer vs computer
python3 pig.py --player1 computer --player2 computer

Timed mode (1-minute max)
python3 pig.py --timed

You can combine options:
python3 pig.py --player1 computer --player2 human --timed
